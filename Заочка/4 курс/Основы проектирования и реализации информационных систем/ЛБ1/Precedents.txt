Администратор:
	
	Название: Удаление предприятия
	Актор: Администратор
	Описание: Администратор заходит в систему, получает список предприятий и удаляет одно из них
	
	Название: Изменение предприятия
	Актор: Администратор
	Описание: Администратор заходит в систему, получает список предприятий и изменяет одно из них
	
	Название: Подтверждение предприятия
	Актор: Администратор
	Описание: Администратор заходит в систему, получает список предприятий и подтверждает одно из них
	
	Название: Получение списка предприятий
	Актор: Администратор
	Описание: Администратор заходит в систему и получает список предприятий
	
Простой пользователь:
	
	Название: Создание предприятия
	Актор: Простой пользователь
	Описание: Простой пользователь заходит в системуь проходит CAPTCHA- и создает предприятие
	
	Название: Прохождение CAPTCHA-теста
	Актор: Простой пользователь
	Описание: Простой пользователь заходит в систему, запрашивает изображение для CAPTCHA-теста и отправляет разгаданный текст
	
	Название: Изменение своего предприятия
	Актор: Простой пользователь
	Описание: Простой пользователь заходит в систему, получает список своих предприятий и изменяет свое предприятие
	
	Название: Получение списка своих предприятий
	Актор: Простой пользователь
	Описание: Простой пользователь заходит в систему и получает список своих предприятий
	
Незарегистрированный пользователь:
	
	Название: Регистриция
	Актор: Незарегистрированный пользователь
	Описание: Пользователь отправляет email и пароль для регистрации и подтверждает email
	
	Название: Подтверждение email
	Актор: Незарегистрированный пользователь
	Описание: Пользователь подтверждает свой email с помощью кода подтверждения email из полученного письма подтверждения
	
	Название: Получение списка подтвержденных предприятий
	Актор: Незарегистрированный пользователь
	Описание: Пользователь запрашивает список подтвержденных предприятий
	
Зарегистрированный пользователь:
	
	Название: Вход в систему
	Актор: Зарегистрированный пользователь
	Описание: Пользователь отправляет email и пароль для входа в систему
